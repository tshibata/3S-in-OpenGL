# 3S-in-OpenGL
An OpenGL Demo with Shadow, Solid and Screen

## Platforms
- glfw
- gtk

## Features
- Shadow mapping
- Solid (normal blend mode) objects
- Lucid (screen blend mode) objects
- Texture mapping
- GLSL
